# @babel/external-helpers-cli

> A command line interface to build Babel external helpers.

See our website [@babel/external-helpers-cli](https://babeljs.io/docs/babel-external-helpers-cli) for more information.

## Install

Using npm:

```sh
npm install --save-dev @babel/external-helpers-cli
```

or using yarn:

```sh
yarn add @babel/external-helpers-cli --dev
```
