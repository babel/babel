# @babel/build-external-helpers

> A command line interface to build Babel external helpers.

See our website [@babel/build-external-helpers](https://babeljs.io/docs/babel-build-external-helpers) for more information.

## Install

Using npm:

```sh
npm install --save-dev @babel/build-external-helpers
```

or using yarn:

```sh
yarn add @babel/build-external-helpers --dev
```
