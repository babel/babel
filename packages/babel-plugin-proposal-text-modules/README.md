# @babel/plugin-proposal-text-modules

> Transform text modules imports (`import '...' with { type: 'text' }`) to work in browsers and Node.js.

See our website [@babel/plugin-proposal-text-modules](https://babeljs.io/docs/babel-plugin-proposal-text-modules) for more information.

## Install

Using npm:

```sh
npm install --save-dev @babel/plugin-proposal-text-modules
```

or using yarn:

```sh
yarn add @babel/plugin-proposal-text-modules --dev
```
